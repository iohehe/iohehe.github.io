Posted on 2020-01-04

0x00 层次聚类描述

  1. 首把每一个样本点看作是一个cluster, 一开始有k个cluster
  2. 根据算法，聚合距离最近的两个cluster，形成k-1个cluster
  3. 循环直至一个最大的cluster形成
  4. 形成一张树状图区分不同的clusters
