Available Expressions Analysis

must analysis

如果说一个expression x op y 在 program point p 上是available的，那么

  1. 从entry到p的all paths必须经过x op y
  2. 每一条path最后的那个evaluation不重新定义x或者y

我理解的evaluation就是动态的经过(计算)
