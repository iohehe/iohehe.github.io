php-src 之编译


0x01 安装

  • 下载： http://php.net/releases/ wget一下，我用的ubuntu16.04 x64

​ tar -zxvf php-7.1.0.tar.gz

  • 进入文件夹分三步:./configure->make->make install

    重点关注的是第一步：

    可以-h看一下选项说明， 这里–prefix是编译安装目录， 打开fpm用–enable-fpm，为了debug加上–enable-debug（debug信息默认-O2, 编译优化选项 ，如果开了debug，make时可以看到 debug为 -O1,开启了编译优化）。

    之后 make install 就将程序安装在指定文件夹下， 其中fpm 放在sbin文件夹内。

在./configure时遇到问题：

error: xml2-config not found. Please check your libxml2 installation.

解决：

sudo apt-get install libxml2-dev

然后重新./configure

配置成功
