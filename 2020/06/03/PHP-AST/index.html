2 child nodes

AST_CALL

函数调用点

  • child0: ARG_LIST - 参数列表

  • child1: 调用函数名

AST_DIM

数组引用

ZEND_AST_DIM = 2 << ZEND_AST_NUM_CHILDREN_SHIFT

  • 0: 数组名

  • 1: 索引值

ZEND_AST_PROP

属性调用

  • 0: objetct_name

  • 1: attr

ZEND_AST_WHILE

  • 0child: condtion
  • 1child: stmt_list

3 child nodes

ZEND_AST_METHOD_CALL = 3 << ZEND_AST_NUM_CHILDREN_SHIFT

  • child0: var:stu
  • child1: name: getScore
  • child2: args:math

ZEND_AST_STATIC_CALL

  • child0: class:stu （直接指向该类）
  • child1: name: getScore
  • child2: args: math

AST_CONDITIONAL

4 child nodes

AST_FOREACH

  • child0: expr: $arr
  • child1: keyVar: $k
  • child2: valueVar: $v
  • child3: stmts: 代码块list (ZEND_AST_STMT_LIST)

ZEND_AST_FOR = 4 << ZEND_AST_NUM_CHILDREN_SHIFT

  • child0: init: $i=0
  • child1: condtion: $i<=1
  • child2: loop: $i++
  • child3: 代码块list (ZEND_AST_STMT_LIST)
